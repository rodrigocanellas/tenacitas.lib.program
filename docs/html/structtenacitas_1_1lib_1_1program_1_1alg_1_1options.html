parses the options passed to a program

Parameters
p_argcnumber of options
p_argvvector of strings with the options
p_mandatoryset of options that are mandatory

an option must be preceded with '–', like '–file-name abc.txt'.

There are 3 types of parameter: boolean, where the parameter has no value, like '–print-log'

single, where the parameter has a single value, like '–file-name abc.txt'

set, where the parameter has a set of values, like '–tests { test1 test2 test3 }

Exceptions
